3. 核心方法分析

调用:

没看源码时，我就纳闷，调用处的k并没有声明，代表的是什么呢？k代表的是入参，new LinkedList<V>()表示返回结果;类似于JS中的Function方法

4. 参数类型通配符extends和super的区别？

由上图可分析得出：

<? extends Integer> 继承了Integer，输入的参数必须是Integer

<? super Integer> 包含了Integer、Number、Object,输入的参数可以是任意类型
