什么是Python?

Python是一种解释型的、面向对象的、带有动态语义的高级程序设计语言。

Python相对Java的优劣之处

优点

  • Python作为动态语言更适合初学编程者，Python可以让初学者把精力集中在编程对象和思维方法上，而不用去担心语法、类型等等外在因素。
  • Python清晰简洁的语法使得它调试起来比Java简单的多
  • Python有一些Java没有的强大的架构，可以表达非常复杂的逻辑；比如Advanced Python Constructs
  • Python有非常强大的支持异步的框架，如Eventlet Networking Library，而Java要实现这些功能要麻烦的多。
  • Python作为脚本语言，更适合开发小的应用，而且极其适合在应用发展初期时用来做原型。

不足

  • 由于是动态语言，所以比Java慢，可以想象一下，Python编译器对于没有类型的变量，需要一个个去check具体是什么类型所消耗的时间，Java相比起C++等等本来就不快，可想而知更慢的效果。
  • Java很适合发展跨平台应用，而Python不适合

Python入门示例

创建列表

Python变量标识符没有类型，且列表可以包含混合类型的数据

遍历列表数据

for循环的结构为for 目标标识符 in 列表 : in将目标标识符和列表分开，冒号表示列表处理开始

定义函数

函数定义的结构def 函数名 (参数) ：，参数表是可选的，但括号不可少，括号后的冒号，表示函数代码组开始

总结

  • 标识符没有类型，用于表示数据对象的名字，但标识符所指示的数据对象有类型
  • 列表可以存放任意数据，且数据可以是混合类型
  • isinstance() 用于判断一个标识符是否是指定的某个类型的数据对象。
  • 用def来定义一个函数
