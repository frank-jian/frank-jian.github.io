锋利的JQuery -- DOM操作

查找节点

操作 描述
var $li = $(“ul li:eq(1)”) 查找元素节点
var p_text = $(“p”).attr(“title”) 查找属性节点

插入节点

操作 描述
B.append(A) 向B内部追加A
A.append(B) 把A追加到B内部
B.prepend(A) 把B内部最前面添加A
B.after(A) 在B后面添加A
A.insertAfter(B) 把A插入B的后面
B.before(A) 在B的前面插入A
A.insertBefore(B) 在B的前面添加A

删除节点

操作 描述
remove() 返回的是删除的元素
detach() 与remove类似，但元素的事件和数据都保留
empty 清空节点，清空内部的对象

复制节点

操作 描述
clone() 复制当前的节点
clone(true) 复制当前的节点以及绑定的事件

替换节点

操作 描述
replaceWith() 替换标签

属性操作

操作 描述
attr(“title”,”xxxx”) 设置参数值
attr(“title”) 获取属性值
removeAttr(“title”) 删除属性值

样式操作

操作 描述
attr(“class”,”xxx”) 设置样式
addClass(“xxx”) 追加样式
removeClass(“xxx”) 移除样式
removeClass() 移除所有样式
toggleClass(“xxx”) 切换样式
hasClass(“xxx”) 判断是否有样式

设置、获取HTML、文本和值

操作 描述
html() 设置或者获取HTML
text() 设置或者获取文本
val() 设置获取值

遍历节点

操作 描述
children() 取得匹配元素的子元素集合
next() 取得匹配元素的下一个同辈元素
prev() 取得匹配元素的前一个同辈元素
siblings() 取得匹配元素的所有同辈元素
parent() 获得集合中每个元素的祖先元素
parents() 获得每个元素的祖先元素
closest() 从本身开始，返回最先匹配的祖先元素

CSS-DOM操作

操作 描述
css(“color”) 获得指定的样式值
css(“color”,”red”) 设置指定样式
width() height() 设置获取宽度和高度
offset() 视窗的偏移值
position() 获取定对于最近的position样式元素的位置
scrollTop 滚动条顶端距离
scrollLeft() 滚动条左端距离
