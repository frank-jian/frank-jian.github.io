Mac技巧

发表于 2016-09-01 | 分类于 Mac |

多任务

手势：三个手指向上滑

快捷键：F3

双屏

操作：鼠标长按绿色按钮

截屏

  • 截取全屏：command + shift + 3
  • 自由截屏：command + shift + 4
  • 所截图图片不存于桌面，而存于剪切板中: command +control + shift + 3
  • 标注截屏重点: 打开图片，点击小工具箱

任务管理器

Mac应用有时会出现无法响应的问题，强制退出没有任何效果，可以使用任务管理器，强制杀死某些进程。

删除文件&清空垃圾箱

删除文件：command + delete

清空垃圾箱: command + shift + delete

浏览器中快速到顶部和尾部

顶部： command + ⬆️

尾部：command + ⬇️

Spotlight 查找文件

快捷键： command + space

休眠

场景：有时候离开座位，又不想让别人看到你电脑上的资料，选择让电脑休眠,特别注意了，此快捷键容易与关机的快捷键command + ctrl + power混淆

快捷键：command + alt + power

去除格式粘贴

场景：有时候复制一段文字，文字中存在超链接，字体等等格式，如果我只想粘贴纯文本信息，怎么办呢？

快捷键: command + shift + V

浏览网页时，快速定位到地址栏

快捷键：command +L

关闭应用程序

场景：以往关闭程序，都是通过双击图标按钮点击退出，发现配合切换程序Commad + Tab来使用关闭程序command + Q效果非常好。

快捷键：command + Q

在Finder中打开当前目录文件

场景：在终端输入 open + . 可以在Finder中开发当前目录

三指拖动文件设置

显示资源库

Mac 10.7系统开始，可能是出于系统安全考虑，默认隐藏用户文件夹下的资源库文件夹

操作：在终端下运行:ln -s ~/Library/ ~/资源库,创建资源库快捷方式

快速呼出emoji表情

快捷键:command + control + 空格键

全屏

场景：全屏和退出全屏是一个平时比较常用的功能

快捷键：command + control + f
