Mysql服务启动错误 The server quit without updating PID file

2016.09.08 流云  热度 ℃ 条评论

背景

本地MySQL一直都能够正常启动的，昨晚关机了Mac，今早启动MySQL,出错了。

解决方案

删除下面这个错误文件就可以了

分析

由于jianweilindeMacBook-Pro.local.err的用户权限是_mysql,而不是我自己的用户权限，故启动的时候会报错。删除这个文件或修改用户权限均可。
