Java IO: 文件

读文件

如果你需要在不同端之间读取文件，你可以根据该文件是二进制文件/文本文件来选择使用FileInputStream/FileReader。这两个类允许你从文件开始到文件末尾读取一个字节/字符，或者将读取到的字节/字符。你不必一次性读取整个文件，你可以按顺序读取文件中的字节和字符。
