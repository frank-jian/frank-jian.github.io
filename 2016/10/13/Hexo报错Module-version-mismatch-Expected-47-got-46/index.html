Hexo报错Module version mismatch. Expected 47, got 46.

解决方案：

  1. 执行npm uninstall hexo卸载hexo；
  2. 重新安装hexo，npm install -g hexo；
  3. 到博客目录执行npm install，hexo clean，hexo generate
