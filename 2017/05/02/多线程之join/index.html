多线程之join

发表于 2017-05-02 | 分类于 JAVA |

thread.join把指定线程加入到当前线程，可以将两个交替执行的线程合并为顺序执行的线程，比如在线程B中调用线程A的join方法，直到线程A执行完毕后，才会继续执行线程B。
