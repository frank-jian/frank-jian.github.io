Sep 2, 2016 | React

React介绍

React 是一个Facebook 和Instagram 用来创建用户界面的JavaScript 库，其Github地址是http://github.com/facebook/react,截至目前已经获取48369个star，说明了React受到了很大的重视。

React主要特点

  1. 作为UI：React可以只作为视图View在MVC中使用，并且在已有项目中，很容易使用React开发新功能
  2. 虚拟DOM:这是React的一个亮点，可以很好地优化视图的渲染和刷新。以前我们更新视图时，需要先清空DOM容器中的内容，然后将最新的DOM和数据追加到容器中，现在React将这一操作放进内存中，React认为内存的操作远比视图全部更新来的高效。
  3. 数据流：React实现了单向的数据流，并相对于传统的数据绑定而言，React更加灵活，便捷。
