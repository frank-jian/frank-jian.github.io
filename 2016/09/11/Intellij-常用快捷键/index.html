Intellij 常用快捷键

2016.09.11 流云 Intellij IDEA  热度 ℃ 条评论

前言

IntelliJ在业界被公认为最好的java开发工具之一,了解其常用快捷键对于工作效率会有一定提升。

场景

查询最近的文件

应用场景：可以快速跳转到之前刚刚查看过的文件。

快捷键：command + E

查看最近编辑的文件

应用场景：可以快速跳转到之前刚刚编辑过的文件。

快捷键：command + shift + E

快速定位到大括号的开头或结尾

快捷键：command + ] 或 command + [

快速查找类

快捷键：command + N

格式化代码

快捷键：command + alt + L

文本替换

快捷键：command + R

全局查找

快捷键： control + shift + F

删除行

快捷键：command + X

复制行

快捷键：command + D

跳到方法实现处

使用场景：通常我们会声明一个接口，然后让一个类实现这个接口。那么怎么从接口，跳到方法实现处呢？可以用这个快捷键

快捷键： command + alt + B

隐藏工具窗口

快捷键： shift + esc

定位到具体某一行

快捷键：command + G

复制当前文件的路径

快捷键：command + shift + C

重命名

快捷键：shift + fn + F6
